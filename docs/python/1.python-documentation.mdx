# Κεφάλαιο 1: Εισαγωγή

Μια γλώσσα προγραμματισμού, όπως και η Python έχει πολλές εντολές, έτοιμες συναρτήσεις/βιβλιοθήκες, κλπ. Όλα αυτά τα εργαλεία έχουν φτιαχτεί για την διευκόλυνση των προγραμματιστών να αναπτύσουν γρήγορα και δομημένα, καλύτερα πρόγραμμα.
Έτσι, η Python, έχει φτιαχτεί για να εκπληρώσει αυτή την ανάγκη.
Ευτυχώς όμως εμείς για να αρχίζουμε να φτιάχνουμε τα δικά μας προγράμματα χρειαζόμαστε λίγα από αυτά.

## Βασικές Έννοιες

### Πρώτο πρόγραμμα

Ας ξεκινήσουμε δημιουργώντας το πρώτο μας πρόγραμμα για να εξοικειωμόμαστε με τις εντολές και τις έννοιες της γλώσσας.
Το πρώτο πρόγραμμα κλάσσικα είναι η εκτύπωση μηνύματος.

```python
print("Γεια σου!")
```

Το αποτέλεσματα του παραπάνω προγράμματος, όπως σωστά μαντέψατε είναι το εξής:

```sh
Γεια σου!
```

:::info Print
Η εμφάνιση ενός μηνύματος προς τον χρήστη γίνεται είτε, με τα διπλά εισαγωγικά (double quotes) `"`, είτε με τα μονά `'`.
Και οι 2 τρόποι είναι ισοδύναμοι. Το πρόγραμμα δεν θα δουλέψει, εάν δεν βάλουμε καθόλου εισαγωγικά!

### Μεταβλητές και Τύποι Δεδομένων

#### Μεταβλητές

Χρησιμοποιούμε διαφορετικούς τύπους δεδομένων, ανάλογα με τις ανάγκες του προγράμματος που θέλουμε να υλοποιήσουμε. Οι μεταβλητές μάς χρησιμεύουν για να αποθηκεύουμε δεδομένα στην μνήμη του υπολογιστή. Η δήλωση μιας μεταβλητής στην Python γίνεται απλά χρησιμοποιώντας το τελεστή `=`.

```python
x = 2025
```

Βασικοί τύποι δεδομένων στην Python είναι οι εξής:

```python
integer = 42  # int - ακέραιος
decimal = 3.14  # float - δεκαδικός
text = "Hello World!"  # string - αλφαριθμητικό
boolean = True  # boolean - λογική
```

:::info Comments

    Στην python μπρούμε να δημιουργούμε σχόλια με το σύμβολο της δίεσης `#`, τα οποία αγνοούνται στην εκτέλεση του προγράμματος. Τα σχόλια που επεκτείνονται σε πολλές γραμμές μπορούμε τα δηλώνουμε με τριπλά `"` πχ.
    ```py
    # This is a single-line comment
    """
    This is a
    multilined comment
    """
    ```

:::

#### Τύποι Δεδομένων

Μπορούμε να δούμε τον τύπο μιας μεταβλητής/τιμής χρησιμοποιώντας την συνάρτηση `type()`.

```py
print(type(4.2))
# <class 'float'>
```

:::info Dynamic Typing

    Στην python μπορούν να γίνουν αναθέσεις στην ίδια μεταβλητή με διαφορετικό τύπο δεδομένων. Αυτό συμβαίνει γιατί δεν χρειάζεται να ορίσουμε τον τύπο δεδομένου μιας μεταβλητής όταν την ορίζουμε.
    ```py
    name = "message"
    name = 13
    name = 3.2
    ```

:::

### Βασικές Πράξεις

#### Αριθμητικές Πράξεις

Η εκτέλεση αριθμητικών πράξεων γίνεται με τους παρακάτω τελεστές:

```python
# Αριθμητικές πράξεις
sum = 10 + 5  # Πρόσθεση
difference = 20 - 7  # Αφαίρεση
product = 4 * 6  # Πολλαπλασιασμός
division = 15 / 3  # Διαίρεση
integer_division = 15 // 4  # Ακέραια διαίρεση
remainder = 17 % 5  # Υπόλοιπο διαίρεσης
exponent = 2 ** 3 # Δύναμη
```

:::info Variable Names

    Μια μεταβλητή μπορεί να περιέχει:

    - Λατινικούς χαρακτήρες a-z, A-Z
    - Αριθμούς
    - Κάτω παύλα `_`

    Επίσης, **ΔΕΝ** πρέπει

    - Να ξεκινάει από αριθμό
    - Να είναι δεσμευμένη λέξη - keyword (πχ. for, if, in, else, κλπ..)

    ```py title="Ονοματολογία Μεταβλητών"
    name1 = "Pickachu" # Έγκυρο
    name_1 = "Pickachu" # Έγκυρο
    _name1 = "Pickachu" # Έγκυρο
    1name = "Pickachu" # Μη έγκυρο
    nam3@ = "Pickachu" # Μη έγκυρο
    ```

:::

#### Πράξεις με strings

Ένας τελεστής μπορεί να έχει διαφορετική συμπεριφορά ανάλογα με τον τύπο δεδομένων που είναι δίπλα του.
Το πιο απλό παράδειγμα είναι η **συνένωση (concatenation)** δύο strings.

```python
word1 = "Easy"
word2 = "Python"
words = word1 + word2
print(words)
# "EasyPython"
```

Στο προηγούμενο παράδειγμα κάναμε μια πράξη με τον τελεστή πρόσθεσης `+` για να ενώσουμε 2 strings. Στην Python μπορούμε με τον ίδιο τρόπο να χρησιμοποιήσουμε τον τελεστή του πολλαπλασιασμού `*` για να επαναλάβουμε ένα string.

```python
word = "he"
word = word * 3
print(word)
# "hehehe"
```

### Έξοδος

Η έξοδος (output) γίνεται πολύ απλά κάνοντας χρήση, όπως έχουμε δει, της συνάρτησης `print()`. Με αυτόν τον τρόπο μπορούμε να εμφανίσουμε μηνύματα προς τον χρήστη και μεταβλητές με αρκετούς τρόπους.

#### Print μεταβλητών

Μπορούμε να εμφανίσουμε στην οθόνη κείμενο και μεταβλητές διαχωρίζοντας κάθε φορά με το χαρακτήρα του κόμματος `,`.

```py
product = "Καφές"
cost = 2.20
print("Το προϊόν είναι", product ,"και κοστίζει", cost, "€" )
```

#### Print με f-strings

Η Python έχει εισάγει έναν εύκολο τρόπο να εμφανίζουμε μηνύματα χωρίς να ανοιγοκλείνουμε εισαγωγικά και να προσθέτουμε κόμματα. Χρησιμοποιώντας τα **f-strings**!

Ένα _f_ μπαίνει πριν τα εισαγωγικά και η μεταβλητή μέσα σε _curly brackets_ `{}`.

```python
name = "Γιώργος"
age = 23
print(f"Γεια σου {name}! Είμαι {age} χρονών.")
# Γεια σου Γιώργος! Είμαι 23 χρονών.
```

Επίσης, οι πράξεις μέσα στις αγκύλες επιτρέπονται κάνοντας τον κώδικα μας πιο ευνάγνωστο.

```python
price = 50
quantity = 3
print(f"Το συνολικό κόστος είναι {price * quantity}€")
# Το συνολικό κόστος είναι 150€
```

### Είσοδος

Όταν θέλουμε να δώσουμε την δυνατότητα στον χρήστη να ορίσει αυτός μια μεταβλητή, χρησιμοποιούμε την συνάρτηση `input()`.

Η **input** παίρνει ως όρισμα ένα string το οποίο και θα εμφανίσει, όταν εκτελεστεί η εντολή. Στην συνέχεια, το πρόγραμμα θα σταματήσει και θα περιμένει από τον χρήστη να ορίσει μια τιμή από το πληκτρολόγιο. Το πρόγραμμα θα συνεχίσει αφού πατηθεί το Enter.

```py
name = input("Πώς σε λένε; ")
print(name)
```

Τέλος, η **input** <u>επιστρέφει</u> αυτό που πληκτρολόγησε ο χρήστης ως string.

:::warning
Εάν θέλετε να εισάγετε ακέραιους ή δεκαδικούς θα πρέπει να μετατρέψετε (**casting**) το string που επιστρέφει η _input_ αντίστοιχα.

- Για παράδεγιμα, εάν θέλουμε να δηλώσουμε τα χρόνια κάποιου:

```python
    years = int(input("Πόσο χρονών είσαι; "))
```

- Εάν πάλι θέλουμε να εισάγουμε δεκαδικό:

```python
    cost = float(input("Πόσο κοστίζει; "))
```

Όπως παρατηρήσατε, χρησιμοποιούμε τις συναρτήσεις _int_ και _float_ για να μετατρέψουμε τα string σε αριθμούς. Παρομοίως, μπορούμε να μετατρέψουμε και έναν αριθμό σε string χρησιμοποιώντας την συνάρτηση `str()`
:::
