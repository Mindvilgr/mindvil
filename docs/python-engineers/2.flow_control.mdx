# 2. Flow Control

Στο προηγούμενο κεφάλαιο, μάθαμε για τις μεταβλητές και τους βασικούς τύπους δεδομένων στην Python.
Η λέξη μεταβλητή σημαίνει "αυτό που μπορεί να αλλάξει" (`μετα + βάλλω`).
Στην πραγματικότητα, όμως, εμείς θέλουμε να αλλάζουμε τη ροή του προγράμματός μας ανάλογα με τις συνθήκες που επικρατούν.

Για παράδειγμα, **αν** βάλετε λάθος PIN στην εφαρμογή της τράπεζάς σας, δεν θέλετε να σας επιτρέψει να μπείτε στον λογαριασμό σας. **Εάν όμως** βάλετε το σωστό PIN, θέλετε να σας επιτρέψει την είσοδο. Εάν κάνετε ανάληψη σε ένα ΑΤΜ, τα χρήματα πρέπει να αφαιρεθούν από το υπόλοιπό σας και το σύστημα πρέπει να σας ενημερώσει για το νέο υπόλοιπο.

Στο παραπάνω παράδειγμα, κάναμε μια υπόθεση (έλεγχος PIN) και ανάλογα με το αν η **υπόθεση ήταν αληθής ή ψευδής**, το πρόγραμμα εκτέλεσε διαφορετικές ενέργειες.

## 2.1 Boolean Logic

### Comparison Operators

Οι τελεστές σύγκρισης μας επιτρέπουν να συγκρίνουμε δύο τιμές και επιστρέφουν μια boolean τιμή (`True` ή `False`).

| Τελεστής | Περιγραφή               | Παράδειγμα          | Αποτέλεσμα |
|----------|------------------------|---------------------|------------|
| `==`     | Ισότητα                | `x == 5`            | `True` αν x είναι ίσο με 5 |
| `!=`     | Ανισότητα              | `x != 5`            | `True` αν x δεν είναι ίσο με 5 |
| `>`      | Μεγαλύτερο από         | `x > 5`             | `True` αν x είναι μεγαλύτερο από 5 |
| `<`      | Μικρότερο από          | `x < 5`             | `True` αν x είναι μικρότερο από 5 |
| `>=`     | Μεγαλύτερο ή ίσο με    | `x >= 5`            | `True` αν x είναι μεγαλύτερο ή ίσο με 5 |
| `<=`     | Μικρότερο ή ίσο με     | `x <= 5`            | `True` αν x είναι μικρότερο ή ίσο με 5 |

### Boolean Operators

Οι λογικοί τελεστές μας επιτρέπουν να συνδυάσουμε πολλαπλές συνθήκες.

|----------|------------------------|---------------------|------------|
| `and`    | Λογικό ΚΑΙ             | `x > 5 and x < 10` | `True` αν και οι δύο συνθήκες είναι αληθείς |

| `not`    | Λογικό ΟΧΙ             | `not(x > 5)`          | Αν η συνθήκη είναι `True`, επιστρέφει `False` και το αντίστροφο |

## 2.2 If Statements

Στην αρχή του κεφαλαίου, αναφέραμε ότι θέλουμε να αλλάζουμε τη ροή του προγράμματός μας ανάλογα με τις συνθήκες που επικρατούν.
Αυτό ακριβώς κάνουν τα `if statements` χρησιμοποιώντας μια **υπόθεση** `if` και ελέγχοντας αν αυτή είναι αληθής ή ψευδής.

### Βασική Δομή If-Else

Η βασική δομή μιας δήλωσης `if-else` αποτελείται από μια **συνθήκη** που αξιολογείται ως `True` ή `False`.
Συνθήκη στην περίπτωση μας είναι μια παράσταση που περιέχει έναν ή περισσότερους τελεστές σύγκρισης και λογικούς τελεστές, όπως αυτοί που είδαμε παραπάνω.

Η σύνταξη για μια βασική δήλωση `if-else` είναι η εξής:

```python
age = 20
if age >= 18:
    print("Είσαι ενήλικας.")
else:
    print("Είσαι ανήλικος.")
```

### If-Elif-Else

Στη πραγματική ζωή, συχνά έχουμε περισσότερες από δύο επιλογές. Για παράδειγμα, αν κάποιος έχει βαθμό 85, θέλουμε να του πούμε "Καλά", αν έχει βαθμό 95, θέλουμε να του πούμε "Άριστα", και αν έχει βαθμό 45, θέλουμε να του πούμε "Ανεπαρκώς".
Αυτό μπορούμε να το κάνουμε με τη χρήση πολλαπλών `elif` (else if) δηλώσεων.

```python
score = 85
if score >= 90:
    print("Άριστα")
elif score >= 75:
    print("Καλά")
elif score >= 50:
    print("Μέτρια")
else:
    print("Ανεπαρκώς")
```

## 2.3 Επαναληπτικές Δομές

Οι επαναληπτικές δομές μας επιτρέπουν να εκτελούμε τον ίδιο κώδικα πολλές φορές, μέχρι να ικανοποιηθεί μια συγκεκριμένη συνθήκη.

### While Loop

```python
count = 0
while count < 5:
    print("Count is:", count)
    count += 1  # Αυξάνουμε το count κατά 1
```

### For Loop

```python
# Επανάληψη σε μια λίστα
fruits = ["μήλο", "μπανάνα", "πορτοκάλι"]
for fruit in fruits:
    print("Φρούτο:", fruit) 
# Επανάληψη σε μια σειρά αριθμών
for i in range(5):  # Από 0 έως 4
    print("Αριθμός:", i)
```

### Break και Continue

```python
# Break: Διακοπή του βρόχου
for i in range(10):
    if i == 5:
        break  # Σταματάει όταν i είναι 5
    print(i)    

# Continue: Παράλειψη της τρέχουσας επανάληψης
for i in range(10):
    if i % 2 == 0:
        continue  # Παράλειψη των άρτιων αριθμών
    print(i)  # Εκτυπώνει μόνο περιττούς αριθμούς
```

## 2.7. Πρόβλημα: Διαχείριση Βαθμολογιών 🤔

Περιγραφή προβλήματος:
Ένας καθηγητής θέλει να διαχειριστεί τις βαθμολογίες των μαθητών του.
Πρώτα, θέλει να υπολογίσει τον μέσο όρο των βαθμολογιών.
Στην συνέχεια, να υπολογίσει τον καλύτερο και τον χειρότερο βαθμό.
Επίσης, θέλει να μπορεί να προσθέτει νέες βαθμολογίες στην λίστα.

# Βαθμολογίες σε διαφορετικές εξετάσεις
grades = [18.5, 16.0, 19.5, 17.0, 20.0]

print(f"Μαθητής: {student_name}")
print(f"Μάθημα: {subject}")
print(f"Βαθμολογίες: {grades}")

# 1. Υπολογισμοί
# Ο ΚΩΔΙΚΑΣ ΕΔΩ! 


print(f"Συνολικές εξετάσεις: {total_grades}")

print(f"Μέσος όρος: {average_grade:.2f}")

# 2. Εύρεση καλύτερης και χειρότερης επίδοσης
# Ο ΚΩΔΙΚΑΣ ΕΔΩ!

# Ο ΚΩΔΙΚΑΣ ΕΔΩ!
